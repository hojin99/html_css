# html_css

## 환경  
node-sass 설치  
npm i node-sass -g  

## 실행  
자동으로 scss를 css로 변환하도록 감시 실행 (package.json의 scripts이용)  
node-sass --watch html/basic/basic.scss html/basic/basic.css  
Live Server를 이용해서 웹브라우저를 실행해서 테스트
