position

정적위치, 상대위치는 웹페이지 흐름에 따라 배치, 상대위치는 웹페이지 흐름에 따른 상대위치, 고정위치는 뷰포트 기준 상대 위치, 절대위치는 상위 Element 기준 상대 위치

정적 위치1(static position)
상대 위치1(relative position)
고정 위치1(fixed position)
절대 위치1(absolute position, zindex)
정적 위치2(static position)
상대 위치2(relative position)
고정 위치2(fixed position)
절대 위치2(absolute position)
